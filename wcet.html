Theory: WCET Response-Time Analysis

Fixed-Point Iteration

The worst-case response time of task τᵢ is computed iteratively:

R⁽⁰⁾ = Cᵢ

R⁽ᵏ⁺¹⁾ = Cᵢ + Σⱼ∈hp(i) ⌈R⁽ᵏ⁾/Tⱼ⌉ × Cⱼ

where hp(i) is the set of tasks with higher priority than τᵢ. Iteration stops when R⁽ᵏ⁺¹⁾ = R⁽ᵏ⁾ (convergence). If R exceeds Tᵢ, the task is infeasible under RMS.

Interference & Blocking

Interference I(τᵢ, τⱼ) is the time stolen from τᵢ's period by higher-priority task τⱼ:

I(τᵢ, τⱼ) = ⌈Tᵢ/Tⱼ⌉ × Cⱼ

Blocking B(τᵢ) is the maximum time τᵢ can be delayed by a single lower-priority task holding a shared resource (priority-inversion blocking). Without Priority Ceiling Protocol, B = max Cⱼ over all lower-priority tasks sharing resources with τᵢ.
